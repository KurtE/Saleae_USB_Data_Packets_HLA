
  # USB Data Packets HLA
  
## Getting started

Warning, this HLA uses my fork and Alpha branch of the usb-analyzer which has not been released

Currently this is not a released HLA for this reason

## what it does

I set up this to work with my version of the usb-analyzer with Frame V2 support (WIP)

This one is specifically so far setup to use that analyzer in Packets mode and extract all of the messages which are data orientated...
